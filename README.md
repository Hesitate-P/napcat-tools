# NapCat Tools Skill

OpenClaw AgentSkill for NapCat QQ API，提供 QQ 消息发送、历史查询、群成员获取、文件操作等功能。

## 功能

- 发送 QQ 消息（私聊/群聊）
- 查询聊天记录
- 获取群成员列表
- 文件上传下载
- 用户信息查询

## 安装

```bash
# 通过 ClawHub 安装
clawhub install napcat-tools

# 或手动安装
git clone https://github.com/Hesitate-P/napcat-tools.git
```

## 使用

在 OpenClaw 中调用 napcat-tools 技能即可使用 NapCat QQ API。

## 依赖

- NapCat QQ 插件
- OpenClaw 框架
